# Speech Practice Application

A Go-based web application to help users improve their speech clarity and fluency through daily practice exercises.

## Features

- **Exercise Library**: Access to 50+ mouth exercises, 100+ tongue twisters, and 20+ diction/pacing strategies
- **Targeted Practice**: Focus on specific sounds (/s/, /z/, /r/, /l/, /th/, /sh/, /ch/, /j/, /k/, /g/)
- **Progress Tracking**: Track practice streaks, total time, and category-specific progress
- **Personalized Recommendations**: Get exercise recommendations based on your performance
- **Accessibility**: High-contrast mode, text size adjustment, keyboard navigation

## Getting Started

```bash
# Run the application
go run cmd/main.go
```

Open http://localhost:8080 in your browser.

## Tech Stack

- Go 1.21+
- Go html/template for server-side rendering
- CSS for styling
